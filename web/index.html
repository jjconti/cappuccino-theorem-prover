Cappuccino

Cappuccino Theorem Prover es un probador automático de teoremas de lógica de predicados de primer orden mediante deducción natural.

Primer release

El primer release del software está planeado para mediados de enero de 2008. Si tenés preguntas, ideas o sugerencias que no pueden esperar, contactate con los autores.
