¿Qué es XXXX?

Es un probador automático de teoremas (ATP - Authomatic Theorem Prober) de lógica de primer orden. Esto significa que pued recibir un conjunto de premisas como:

Premisas..

y objetivos

objetivos

y obtener la secuencia de pasos que tendría que realizar un humano para demostrar que los objetivos son consecuencia lógica de las premisas.

Uso

Desarrollo

El control de versiones del código fuente es llevado a cabo en un repositorio svn. Puede obtener una copia del codigo fuente mediante el siguiente comando en GNU/Linux

svn co http://svn.juanjoconti.com.ar/logic/

o en Windows mediante alguna de muchas herramientas. La misma dirección también permite navegar por el código fuente: http://svn.juanjoconti.com.ar/logic/
